# 运营分析平台

## 项目在线预览地址
[http://tools.crofys.cn/](http://tools.crofys.cn/)


## 注意事项

> node 版本 v11.9.0
> npm 版本 6.5.0

## 项目技术栈

> vue vue-router vuex element-ui node webpack

## 执行命令

### 1. 克隆项目
```
git clone https://github.com/xcj189323200/tools-manage.git
```
### 2. 切换分支
```
git checkout dev
```
### 3. 下载 npm 包
```
npm i  --registry=https://registry.npm.taobao.org
```
### 4. 本地开发
```
npm run dev
```
